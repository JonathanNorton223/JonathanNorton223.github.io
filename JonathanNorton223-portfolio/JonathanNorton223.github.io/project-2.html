Project Overview

This project highlights database programming and application design using Python and MongoDB. It focuses on creating reusable CRUD functionality, connecting application logic to stored data, and presenting information in a way that supports decision-making.

The work demonstrates modular programming, database connectivity, query filtering, and maintainable code practices that can be adapted for future dashboard, API, or data-management projects.

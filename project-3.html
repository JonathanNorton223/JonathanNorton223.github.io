CS 370 - Current/Emerging Trends in Computer Science

Artificial intelligence and emerging technology project focused on machine learning, ethical computing, cybersecurity, and the impact of modern computing trends.

Project Overview

This project explored current and emerging trends in computer science, including artificial intelligence, machine learning, cybersecurity, data privacy, and ethical technology development.

The work focused on understanding how new technologies affect software systems, organizations, users, and society. It also examined the responsibilities developers have when working with intelligent systems and sensitive data.

This project demonstrates my ability to evaluate modern computing trends, analyze technical and ethical challenges, and explain how emerging technologies can be used responsibly in software development.
